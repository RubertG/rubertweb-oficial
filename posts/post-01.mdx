El Tres en Raya es un juego sencillo que todos conocemos. En este artículo exploraremos loa mas importante sobre la lógica, la funcionalidad y la interfaz de este juego. Se utilizaron tecnologías como React, Tailwind y TypeScript. Es fundamental tener en cuenta que este proyecto se inició utilizando la herramienta <a href="https://es.vitejs.dev/" target="_blank">Vite</a>.

<hr />

## Estructura de carpetas del proyecto

Tomando la estructura que nos proporciona <a href="https://es.vitejs.dev/" target="_blank">Vite</a> al crear un proyecto, se organizó en distintas carpetas, lo que dio como resultado la siguiente estructura.

<div data-language='sh'>

## Constantes

Para el desarrollo de la lógica y funcionalidad hay varias constantes importantes, estas son: 
- `OPTIONS{:ts}` <span> Las opciones de juego **X** y **Y**.</span>
- `WINNIG_COMBO{:ts}` <span> Las combos ganadores del juego de juego. Para representar el tablero vamos a utilizar una matriz y cada número será la posicion de la opción **X** o **Y**.</span>

<div data-language="ts">
  ```ts
    export const OPTIONS = {
      X: 'X',
      O: 'O'
    } as const

    export const WINNING_COMBO = [
      [0, 3, 6],
      [1, 4, 7],
      [2, 5, 8],
      [0, 1, 2],
      [3, 4, 5],
      [6, 7, 8],
      [0, 4, 8],
      [2, 4, 6]
    ] as const
  ```
</div>
